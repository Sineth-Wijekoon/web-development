Home Page

Technical Description

HTML Structure and Key Elements

Navigation Bar
  • The header tag contains the logo and the navbar, which makes up the whole navbar
  • The Navbar is created using an unordered list to create a horizontal menu
  • The navigation bar provides quick access to different pages in the website seamlessly
The Logo
  • The homelogo_container contains and img tag and an h1 tag
  • Broken down into two classes one for the image and one for the heading
  • This is used to present the user with the logo and an immediate insight into the websites goal
Quote Section
  • Uses span for a text inside a textbox and img for an image to describe the text
  • Engages the users emotionally and provides a fact, making users feel sympathatic
Horizontal Scrolling Section
  • Horizontal scrolling of 3 images as the user scrolls down
  • The functionality is created using javascript
  • 3 images scroll horizontally to enhance user experience
Statistics Section
  • Uses an unordered list to display key statistics horizontally
  • Visually emphasizes key numbers related to hunger
Causes Section
  • img tag on the left and an h2 and ordered list on the right
  • Describes some of the causes for global hunger in a list and an image is added for visual enhancement
  • Coloured border for card like effect
Mission Section
  • Two coloumn layout with an image from the SDG and a brief description of the website mission
Team Cards
  • Each card contains an image,title and description related to each members content pages
  • Clicking on each card will take you to the relevant members content page
Footer
  • Contains a link to the authors editor page

CSS Techniques and Design Choices

Navbar Styling
  • Flexbox (display: flex; justify-content: space-between;): Ensures a responsive navigation menu.
  • Navbar a:hover is used to create a light yellow box around the current hovered link to enhance user experience
  • Havbar a:active is used to make the yellow box remain around the website the user is currently on
Image Styling
  • border-radius has been used to soften the corners of images for a modern look
  • box-shadow creates a shadow around each image
  • transition: (on hover) has been used to add an interactive zoom effect as the user hovers on the image
Quote Section Styling
  • Large font size ensures the key message stands out
  • The background colour used creates a strong contrast to highlight the content
Sticky Section Styling
  • position: sticky; top: 0;: Keeps the section fixed while scrolling.
Statistics Section Styling
  • Flexbox display:flex aligns items horizontally
  • The colour contrast highlights numbers to draw attention
Causes Section Styling
  • (margin-left: 700px; margin-top: -300px;): Aligns the text box over the image.
  • border-top and border-right creates a frame-like effect
Visual Styling
  • Use of a variety of headings make the important topics stand out
  • Hover effects improve user engagements

Accessibility

Aria Roles and Labels
  • Use of roles to assist with assitive techologies and labels provide a clear description or purpose for a section
  • role="navigation" defines the navigation bar for assistive technologies.
  • aria-label="Main Navigation Bar" provides a clear description for the navigation region.
Semantic HTML Usage
  • Proper use of tags like header section and figure improves the structure of the code
  • use of figure and figcaption for images with descriptive texts
Image Accessibility
  • Included alt attributes to all image tags to explain the purpose of the buttons
Text readability
  • Use of h1 - h6 heading types to have a hierachy and to maintain a logical flow
Links
  • Use of descriptive link names for user easiness

Table Page

Technical Description

HTML Structure and Key Tags

Table Container
  • Provides a container for the entire table
  • The use of a container helps in creating a structured design, and allows stlying to be applied to the entire page
Table tag
  • The table tag is used to represent the data in a tabular format, it contains all the rows and columns
  • The id-statistics table is used to identify the table in css
tr, td and th Tags
  • tr tag is used to define a table row
  • td tag is used to represent the data in a single cell
  • th tag represents table headings and used to emphasize headings of a certain column

CSS and Style

border-collapse: collapse
  • Used to remove the gaps between the table borders
  • Makes the table look more compact
Rowspan & Colspan
  • Used to merge multiple rows/columns together
  • The table topic is merged into 4 four columns giving it more prominence
Position Relative and Absolute
  • Position Relative is used to establish it as a containing element for its children element
  • Position Absolute is used to position an element relative to its parent container
Font-Size and Background Colour
  • font-size is used to adjust the font size of the text within the cell
  • Background Colour is used to adjust the colour of the cell
:hover
  • :hover is used to adjust css so that it changes to specified properties on hover

Accessibility

  • I have used alt text in images to display alt text incase the image doesnt appear
  • Used th scope="col" helps assistive technolgies assovciate with respective columns
  • Contrasting Colours for better readability
